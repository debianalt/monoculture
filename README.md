# The contingent cooperative

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19369616.svg)](https://doi.org/10.5281/zenodo.19369616)

**State infrastructure and the juridical-form composition of new organisations in Misiones, Argentina, 1901–2025**

Raimundo Elías Gómez · María Gabriela Miño
CONICET / FHyCS-UNaM, Posadas, Argentina

---

## Overview

This repository contains the data and code for a study of the formal organisational landscape of the Argentine province of Misiones (1901–2025). The analysis assembles 14,169 dated formal organisations and characterises — with model-free, descriptive measures — how the juridical-form composition of new registrations tracks the legal and fiscal instruments of a pre-1990 baseline and the seven national state projects since 1990.

The central finding is a contingency asymmetry: self-sustaining commercial forms (most recently the *Sociedad por Acciones Simplificada*, SAS) diffuse secularly across state projects of opposed political orientation, whilst the cooperative, whose creation depends on a continuous social-economy apparatus, surges when that apparatus is sustained and reverts to its long-run baseline when it is withdrawn.

## Key findings

- The SAS rose monotonically from 8.2% to 19.7% to 43.9% of new registrations across the Macri, Fernández, and Milei projects — a secular diffusion independent of any single project's programme.
- Cooperative registration tracked the social-economy apparatus: 68–93 per year under the Kirchnerist and Fernández projects, reverting to 9 per year under Milei — identical to the Macri rate. The decline (−90%) is more than twice as steep as the contraction of the registration pool (−39%), the opposite of what a counter-cyclical account of cooperative entry predicts.
- Cooperatives registered whilst the apparatus operated show a weaker subsequent presence in the federal tax system, relative to the commercial organisations registered alongside them, than those registered before it existed.
- The shift is neither spatially uniform (SAS modal in 46% of active census radios, within the permutation null, *p* = 0.49) nor aligned with forest cover (*r* = 0.056, *N* = 17, unstable under jackknife).
- Juridical-form diversity declined under the most recent project but is statistically indistinguishable from its pre-1990 level (*H* = 1.44, 95% CI [1.35, 1.51] against 1.43 [1.40, 1.46]).

## Repository structure

```
monoculture/
  data/        Datasets (see data/README.md for the dictionary)
  code/        Analysis scripts
  tables/      Canonical output tables (CSV), regenerated by analysis_diversity.py
  figures/     Publication figures (300 DPI)
  archive/     Superseded MCA/sequence pipeline (see archive/README.md)
```

The deposit is self-contained: every script reads from `data/` and writes to `tables/` or `figures/` relative to the repository root, so a clone reproduces every number in the paper without any external path.

## Data

| File | Records | Description |
|------|---------|-------------|
| `geocoded_sociedades.parquet` | 14,177 | Geocoded organisational registry — Misiones subset of the *Registro Nacional de Sociedades* |
| `enriquecido_arca_v2.parquet` | 14,278 queried | ARCA fiscal-status enrichment (6,652 with an active record) |
| `lookup_localidad_departamento.csv` | 283 | Locality-to-department mapping |

Source: *Registro Nacional de Sociedades*, published by the Argentine Ministry of Justice on the national open-data portal and compiled from ARCA fiscal records; retrieved 23 February 2026, filtered to organisations whose registered fiscal domicile lies in Misiones. See `data/README.md` for the full dictionary.

## Code

| Script | Description |
|--------|-------------|
| `01_enrich_base.py` | Enrich the organisational registry with ARCA fiscal data |
| `02_geocode_sociedades.py` | Geocode fiscal addresses via Nominatim + OSM |
| `analysis_diversity.py` | **Canonical analysis**: Shannon diversity with bootstrap intervals, composition with exact Poisson intervals, cooperative subtypes, sub-departmental dominance with a permutation null, and fiscal status by form and period |
| `make_figures.py` | Publication figures (Fig2–Fig5) from the canonical tables |
| `11_study_area_map.py` | Study-area map (Fig1) |

Every quantity reported in the paper is computed by `analysis_diversity.py` on the raw juridical form. No geometric or sequence method is used: an earlier pipeline entered the period of creation as an active variable in a correspondence analysis, which makes the principal axis an axis of time and manufactures a spurious terminal convergence. That pipeline, and the diagnostics establishing why it was abandoned, are retained in `archive/` (see `archive/README.md`); no result in the paper is computed from them.

## Requirements

Python 3.10+. Install dependencies:

```bash
pip install -r requirements.txt
```

Then, from the repository root:

```bash
python code/analysis_diversity.py   # regenerates tables/
python code/make_figures.py         # regenerates figures/Fig2–Fig5
```

The department-level forest-cover check in `analysis_diversity.py` requires a local PostGIS database and is skipped when it is unavailable; every other table is reproduced from the deposited data alone.
